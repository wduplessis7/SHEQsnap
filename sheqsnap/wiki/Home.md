# Welcome to the SHEQsnap Wiki

**SHEQsnap** is a web-based Safety, Health, Environment, and Quality (SHEQ) management system designed to help organisations capture, track, and manage near misses, incidents, corrective actions, and safety log entries — all in one place.

This wiki is the complete reference guide for SHEQsnap users, administrators, and safety officers.

## Quick Start Summary

1. Install and run SHEQsnap (see [Getting Started](Getting-Started))
2. Log in with the default Admin credentials
3. Create your departments, sites, and groups under **Admin**
4. Add users and assign roles
5. Start logging near misses and incidents
