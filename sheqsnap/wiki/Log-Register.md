# Log Register

---

## Purpose of the Log Register

The Log Register is used to record all formal safety documentation that does not fit into near misses or incidents. This includes inspections, toolbox talks, meeting minutes, permits, and other evidence of safety activities. The Log Register helps organisations demonstrate compliance and maintain a complete safety file.

---

## Log Types

| Log Type | Description |
|----------|-------------|
| **Inspection** | A formal site or equipment inspection report |
| **Toolbox Talk** | A pre-shift safety briefing with sign-off by attendees |
| **Meeting Minutes** | Minutes from a health and safety committee or team meeting |
| **Safety File** | A complete safety file document (often required by law for contractors) |
| **Permit** | A permit to work (e.g., hot works, confined space, working at height) |
| **Incident Log** | A supplementary log entry linked to an incident record |
| **Other** | Any other document that needs to be captured in the safety register |

Log types can be managed by an Admin in **Admin > Log Types**.

---

## Reference Number Format

Each log entry is automatically assigned a unique reference number in the format **LOG0001**, **LOG0002**, etc.

---

## How to Create a Log Entry

1. Click **Log Register** in the sidebar
2. Click **+ New Log Entry**
3. Select the **Log Type**
4. Fill in the title, description, date, department, and company (if applicable)
5. Upload any relevant documents (see Uploading Attachments below)
6. Click **Save as Draft** or **Submit for Approval**

---

## Field Descriptions

| Field | Description | Required |
|-------|-------------|----------|
| **Title** | Short description of the log entry | Yes |
| **Log Type** | Category of the log (see Log Types above) | Yes |
| **Date** | Date the activity took place | Yes |
| **Department** | The department this log relates to | Yes |
| **Company** | The contractor company, if this log is for a contractor | No |
| **Description / Notes** | Full details, notes, or summary of the activity | No |
| **Attendees** | Names of people present (for toolbox talks and meetings) | No |
| **Attachments** | Supporting documents, sign-off sheets, photos | No |

---

## Linking to a Company or Department

Log entries can be linked to a specific company (useful for contractor documentation) and a department:

1. In the **Department** field, select the relevant department
2. In the **Company** field, select the contractor company (if applicable)

This allows filtering and reporting by company or department.

---

## Uploading Attachments to a Log Entry

1. Open or create a log entry
2. Scroll to the **Attachments** section
3. Click **Upload File**
4. Select your file (PDF, Word, Excel, JPG, PNG, etc.)
5. Click **Upload**

Multiple files can be attached to a single log entry. All attachments are stored securely and can be downloaded at any time.

---

## Log Entry Statuses

| Status | Description |
|--------|-------------|
| **Draft** | Entry has been saved but not yet submitted |
| **Pending Approval** | Entry has been submitted and is awaiting approval from a Safety Officer, Manager, or Admin |
| **Active** | Entry has been approved and is active in the register |
| **Archived** | Entry is no longer current but is retained for record-keeping |

---

## Approval Process for Log Entries

Log entries can be submitted for approval when they are ready for review:

1. Create or edit a log entry
2. Click **Submit for Approval**
3. The entry status changes to **Pending Approval**
4. A Safety Officer, Manager, or Admin receives an alert in the Approvals queue
5. They review the entry and either **Approve** or **Reject** it
6. If approved, the status changes to **Active**
7. If rejected, a reason is provided and the entry goes back to the submitter for correction

---

## How Contractors' Log Entries Go to Pending Approval Automatically

When a user with the **Contractor** role creates a log entry, the status is automatically set to **Pending Approval** — even if they do not manually submit it. This ensures that all contractor documentation is reviewed by an internal team member before it becomes active in the register.

---

## Viewing and Downloading Attachments

1. Open any log entry in the register
2. Scroll to the **Attachments** section
3. Click on the file name to download or preview it

Attachments are available to all users who have access to that log entry. Only users with edit permissions can delete attachments.
